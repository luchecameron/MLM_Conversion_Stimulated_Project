# MLM Lead Conversion Analysis & Power BI Dashboard

## Project Overview


This is a **simulated end-to-end data analytics project** created to replicate a real-world business scenario involving Multi-Level Marketing (MLM) lead conversion analysis.

The project was developed using a mock dataset designed to represent lead activity generated through social media platforms. The objective was to analyze conversion behavior, identify key drivers influencing successful lead conversions, and provide data-driven recommendations.

The project combines **Python-based exploratory data analysis (EDA)** with an **interactive Power BI dashboard** to transform raw data into actionable insights and business-focused reporting.

This simulated case study was designed to mirror a real analyst workflow, including:

- Business requirement analysis
- Data cleaning and preparation
- Exploratory Data Analysis (EDA)
- Visualization and storytelling
- Dashboard development
- Business recommendations

The goal was to recreate a realistic data analytics process and demonstrate technical and business problem-solving skills.
---

## Business Problem

The business generates leads from multiple social media platforms such as Instagram, Facebook, and TikTok. Although many users show interest in products and services, conversion rates remain lower than expected.

This analysis investigates lead behavior and engagement patterns to answer key business questions and identify opportunities to improve conversion outcomes.

---

## Project Objectives

- Determine the overall conversion rate of MLM leads
- Identify key factors influencing conversions
- Analyze lead behavior across multiple segments
- Evaluate response times and follow-up effectiveness
- Discover trends and conversion patterns
- Provide actionable business recommendations
- Present findings through an interactive dashboard

## Dataset Features

| Feature | Description |
|----------|-------------|
| lead_id | Unique lead identifier |
| age | Lead age |
| gender | Lead gender |
| platform | Social media platform |
| ad_type | Advertisement type |
| interest_level | Lead engagement level |
| time_to_contact_hours | Hours before contact |
| num_followups | Number of follow-ups |
| product_category | Product category |
| lead_source | Organic or Paid |
| converted | Conversion status |
